# 🚀 Wispotter (MyIdentityEmail) - Gelişmiş Mail Yönetim Sistemi

Wispotter, **ASP.NET Core** mimarisi üzerine inşa edilmiş, kullanıcıların gerçek zamanlı mail alışverişini yönetebildiği, güvenlik ve performans odaklı bir web panelidir. Bu proje, modern bir email servisinin sahip olması gereken tüm temel ve ileri düzey özellikleri barındırır.

## ✨ Öne Çıkan Özellikler

* **ASP.NET Identity Entegrasyonu:** Güvenli kayıt, giriş ve rol bazlı yetkilendirme süreçleri.
* **MimeKit & MailKit:** SMTP üzerinden gerçek mail gönderimi ve Inbox (Gelen Kutusu) yönetimi.
* **E-Posta Onay Sistemi:** Gmail App Password entegrasyonu ile hesap doğrulama kodları.
* **Zengin Metin Editörü:** CKEditor entegrasyonu ile profesyonel mail içerikleri oluşturma.
* **Dinamik Sidebar & Dashboard:** CSS3 efektleri ve BoxIcons ile modernize edilmiş kullanıcı arayüzü.
* **Gelişmiş Klasörleme:** Gelen Kutusu, Gidenler, Taslaklar ve Yıldızlı Mesajlar sekmeleri.

---

## 🛠 Kullanılan Teknolojiler

* **Backend:** .NET 8/9, ASP.NET Core MVC
* **Veritabanı:** MS SQL Server & Entity Framework Core
* **Güvenlik:** ASP.NET Identity, SMTP Authentication
* **Frontend:** HTML5, CSS3 (Advanced Animations), Bootstrap, jQuery
* **Araçlar:** CKEditor, MimeKit, MailKit
